# babel-js

Project to practice babel, browserify and typescript.

## Project Setup

Use the package manager [npm](https://www.npmjs.com/)

```bash
npm install
npm run build
npm start
```

## Dependencies

* npm 6.13.4
* http-server 0.12.3
* babel 7.10.5
* browserify 16.5.2
* tsify 5.0.1
* typescript 3.9.7
